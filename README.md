# AI-based-ppt-analyzer

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Piyus563/AI-based-ppt-analyzer)

AI PPT Analyzer is a Flask web app for uploading `.pptx` files, analyzing presentation quality, exporting PDF reports, and downloading corrected presentations.

## Deploy

This project includes `render.yaml`, so it can be deployed on Render as a Python web service.
